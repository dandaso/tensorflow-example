## 環境設定

- Anacondaをインストールする
```
python --version
Python 3.5.2 :: Anaconda 4.2.0 (x86_64)
```
- AnacondaにTensorFlowをインストールする  
```
# 仮想環境 tf内にインストール
conda create -n tf python=3.5.2
source $PYENV_ROOT/versions/anaconda3-4.2.0/bin/activate
conda install -c conda-forge tensorflow
```
